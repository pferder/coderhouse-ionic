- Maquina Virutal con Android
	- Instalar en este orden
	- [VirtualBox](https://www.virtualbox.org/)
	- [Vagran](https://www.vagrantup.com/)
	- [Ionic Box](https://github.com/driftyco/ionic-box)

- Si quieren bajar Convergence
	- Clona o bajar el repo (https://github.com/GrumpyWizards/Convergence)
	- Entran a la carpeta donde bajaron el repo con la consola e instalan las dependencias con el comando "npm install"
	- Corren el comando "ionic serve"

- Manejador de versiones de Node 
	- Instalar n "npm install -g n" https://www.npmjs.com/package/n
	- Correr comando "n stable"
